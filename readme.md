Before you proceed, ensure you have the following prerequisites installed:

## Setup, Run, and Contribute to the Project

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/dreamer-coding-555/xtest-runner.git
   cd xtest-runner
   ```

2. **Run the Project**:
   ```bash
   python xrunner.py
   ```
